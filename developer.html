Skills & Learning Progress

What I'm currently learning and discovering through this first project

Frontend Development Basics

Just learning the fundamentals of HTML5, CSS3, and JavaScript! Still figuring out how to structure web pages properly, style them to look nice, and add basic interactivity. Every tag and CSS property is a new discovery in my coding journey.

Learning Responsive Design

Trying to understand how to make websites work on different screen sizes! CSS Grid and Flexbox are still confusing, but I'm slowly learning how media queries work. Making things look good on both phones and computers is trickier than I thought!

Discovering Data Analytics

Chart.js is amazing but overwhelming! I managed to copy some examples and modify them to create basic charts for this project. Data visualization seems really powerful, but I'm still learning what all those numbers and graphs actually mean.

Learning CRUD Operations

CRUD stands for Create, Read, Update, Delete - I just learned that! The admin panel was my first attempt at letting users add, edit, and remove items. localStorage is pretty cool for storing data in the browser, even though I'm sure there are better ways to do it.

Trying UI/UX Design

I'm learning that making websites look nice is harder than it seems! Picking colors that work together, making buttons look clickable, and organizing information so it makes sense - there's so much to consider. Most of my design choices are based on "does this look okay?" rather than expertise.

Learning to Organize

This project taught me that keeping files organized is super important! I'm learning to name my CSS classes properly, comment my code so I remember what it does, and keep track of all the different files. Still figuring out best practices, but getting better!
